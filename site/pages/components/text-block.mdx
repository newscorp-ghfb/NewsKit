# TextBlock

## Overview

TextBlock is a paragraph, which can be used to summarize any text content. It can be in larger or bolder type than regular paragraph text and this is typically to attract the userâ€™s attention to read more of the article.

### Usage

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="children" type="string" required>
    The content of the TextBlock is passed as the child of the component.
  </Prop>
  <Prop
    name="as"
    type="'h1' | 'h2' | 'h3' | 'h4' | 'h5' | 'h6' | 'div' | 'span'"
    default="p"
  >
    This is a string name with one of the following html elements: h1, h2, h3,
    h4, h5, h6 or div, that the TextBlock can be converted to. TextBlock
    defaults as a p element, but it can be converted to another heading element
    or div by `as="div"` prop.
  </Prop>
  <Prop name="noCrop" type="boolean">
    If true, this disables text cropping functionality on the TextBlock.
  </Prop>
  <Prop name="stylePreset" type="MQ<string>">
    If provided, this overrides the style preset applied to the TextBlock.
  </Prop>
  <Prop name="typographyPreset" type="MQ<string>">
    If provided, this overrides the typography preset applied to the TextBlock.
  </Prop>
</LegacyBlock>
