# Unordered List

## Overview

### Usage

An unordered list can be used to produce a list of related content prefixed with no bullets as default, this can be optionally overridden with a custom icon.

## Props

<LegacyBlock marginTop="sizing040">
    <Prop name="children" type="Array<React.ReactNode>" required>
    An array of react nodes to populate the list components.
    If the node is a react element an unique key must be given to each element within the array.
  </Prop>
  <Prop name="listItemMarker" type="IconComponent" default='IconFilledCircle'>
    Optional icon component to override the list style.
  </Prop>
  <Prop name="markerAlign" type="'start' | 'center' | 'end'" default='center'>
    Vertically align the listItemMarker when the list takes more than one line.
  </Prop>
  <Prop name="overrides" type="object">
    If provided, overrides the respective presets for the component and provided elements.
  <Prop name="spaceStack" type="MQ<string">
    If provided, this overrides the space between the list items.
  </Prop >
  <Prop name="content">
    <Prop name="stylePreset" type="MQ<string>">
      If provided, this overrides the style preset applied to content.
    </Prop>
    <Prop name="typographyPreset" type="MQ<string>">
      If provided, this overrides the typography preset applied to content.
    </Prop>
  </Prop>
   <Prop name="marker">
    <Prop name="stylePreset" type="MQ<string>">
      If provided, this overrides the style preset applied to the marker.
    </Prop>
    <Prop name="spaceInline" type="MQ<string>">
      If provided, this overrides the space between the marker and the content.
    </Prop>
    <Prop name="size" type="string">
      If provided, this overrides the marker icon size.
      <br />
      <br />
      <i>Note: You can also set the icon size by passing it directly as a size prop to the icon,
      but by doing this you will override the iconSize passed from overrides.
      We discourage this approach but we will keep the functionality
      in case it is needed.</i>
    </Prop>
  </Prop>
  <CommonLogicalPropsMDX nested />
Refer to the defaults below for the full object structure:

  <CodeFromDefaultPresets componentName="unorderedList" />
  </Prop>

</LegacyBlock>
