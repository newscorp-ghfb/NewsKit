# Button

## Overview

A styled button element with customisable size.

## Props

Buttons have a number of props to facilitate a variety of uses. Any prop valid on an Button HTML element as described in <a href='https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#Attributes)'>here</a>.
Custom props are detailed below:

### Styling

The button & Icon Button components are block level elements with a native HTML button with custom styling.

### Accessibility

Each are fully accessible with the native HTML button used.

#### Colour

Text and background colours must have a 3:1 minimum contrast ratio.
Hover states should have clear contrast change or should include a non colour designator.

## Button as link

Buttons have the ability to render as links when a value for the href prop is passed.
This is suitable in cases when you may want the styling of a button but the behaviour of a link.

Any prop valid on an [Anchor HTML element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a#Attributes) is also valid on the Button link component

<Code>
  {`
  <Button href="www.newskit.co.uk" overrides={{stylePreset: 'buttonOutlinedPrimary'}}>
    Click me!
  </Button>
`}
</Code>
<Code>
  {`
  <IconButton aria-label="Link icon" href="https://www.newskit.co.uk/">
    <IconFilledLink />
  </IconButton>
  `}
</Code>

## Variations

Buttons are cues that trigger actions. They are foundation components that allow users to perform actions or navigate an experience in our products.

NewsKit provides 5 types of predefined style preset button variants and corresponding icon buttons, each representing a specific user intention or visual cue that can be performed by the user.

### Primary

Intended as the leading trigger or cue of an action. Visually should be the most dominant visual cue used on a page, and as such should be used sparingly to ensure visual balance in the page hierarchy.

Add a button primary style preset token (buttonSolidPrimary, buttonOutlinedPrimary, buttonMinimalPrimary, iconButtonSolidPrimary, iconButtonOutlinedPrimary, iconButtonMinimalPrimary) to the overrides .

| Type     |                                     Button                                     |                                                        Icon Button                                                         |
| -------- | :----------------------------------------------------------------------------: | :------------------------------------------------------------------------------------------------------------------------: |
| Solid    |  <Button overrides={{stylePreset: 'buttonSolidPrimary'}}> Click me </Button>   |  <IconButton aria-label="Link icon" overrides={{stylePreset: 'iconButtonSolidPrimary'}}><IconFilledEmail /></IconButton>   |
| Outlined | <Button overrides={{stylePreset: 'buttonOutlinedPrimary'}}> Click me </Button> | <IconButton aria-label="Link icon" overrides={{stylePreset: 'iconButtonOutlinedPrimary'}}><IconFilledEmail /></IconButton> |
| Minimal  | <Button overrides={{stylePreset: 'buttonMinimalPrimary'}}> Click me </Button>  | <IconButton aria-label="Link icon" overrides={{stylePreset: 'iconButtonMinimalPrimary'}}><IconFilledEmail /></IconButton>  |
